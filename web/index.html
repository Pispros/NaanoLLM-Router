info

Heads up: this tab launches each engine with a minimal, portable command and no hardware tuning by default (GPU offload, KV-cache quantization, context size, tensor-parallel, slot persistence…). You can add those flags yourself in the “Extra launch args” field — they are passed verbatim to the command. For full control, run each engine as your own service or script tuned to your hardware and context, then just point the Endpoints tab at its base URL — you can skip this tab entirely.
